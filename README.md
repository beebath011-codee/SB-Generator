# ISP Config Generator

A web-based tool for IT Network Engineers to quickly generate ISP customer configuration commands.

## Features
- PPPoE and Static IP configuration modes
- Dark (hacker) / Light (professional) theme toggle
- Auto-parse customer info from pasted text
- One-click copy to clipboard
- VLAN mapping for multiple service types
